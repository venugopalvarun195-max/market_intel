LNG full-chain logic

1. Purchase at wellhead (Henry Hub / netback)
2. Add liquefaction tolling — biggest single cost (~$2–3)
3. Add shipping: voyage + BOG fuel (~0.10–0.15%/day)
4. Add regas terminal fee at destination
5. Net = destination price − all costs. Arb open if > 0

Crude arb logic

1. Buy at origin: benchmark ± grade differential
2. Freight via Worldscale: flat rate × WS points ÷ 100
3. Quality adj: API gravity + sulphur shift refinery yield
4. Sell at destination benchmark (Brent / Dubai / WTI)
5. Net = dest value − origin cost − freight − adj
