LNG Full-Chain Logic

1. Purchase at wellhead — Henry Hub / netback pricing
2. Liquefaction tolling — biggest cost (~$2–3/MMBtu)
3. Shipping: voyage charter + BOG fuel (~0.10–0.15%/day)
4. Regas terminal fee at destination
5. Net = destination price − all costs. Arb open if positive
Key drivers: JKM–HH spread, VLCC/LNG carrier freight, Panama Canal optionality

Crude Arb Logic

1. Buy at origin: benchmark ± grade differential
2. Freight via Worldscale: flat rate × WS points ÷ 100
3. Quality adj: API gravity + sulphur shift refinery yield value
4. Sell at destination benchmark (Brent / Dubai / WTI)
5. Net = dest value − origin cost − freight − adjustments
Key drivers: Brent–Dubai EFS, VLCC rates, grade differentials
